## How to Add Images:

### Method 1: WordPress Media Library
1. Go to WordPress Admin → Media → Add New
2. Upload your images
3. Copy the image URLs
4. Update the theme files with the correct URLs

### Method 2: FTP Upload
1. Upload images directly to `/wp-content/themes/zero-to-nine-marketing/assets/images/`
2. Images will automatically work

### Method 3: Elementor (If you want to use Elementor)
1. Install Elementor plugin
2. Edit pages with Elementor
3. Add Image widgets where needed
4. Upload images through Elementor interface
